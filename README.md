# idiom-puzzle

#### 介绍
一个简易的成语接龙网站

#### 软件架构
B/S架构（Vue + SpringBoot）
使用成语库文件为基础，通过jpinyin模块进行拼音解析


#### 安装教程

1.  不需要数据库，server模块通过maven打jar包即可使用
2.  推荐jar包启动参数 -Xms100m -Xmx100m

#### 使用说明

1.  打开网页：http://ip:8080
2.  在输入框中输入成语（或者一个汉字），然后点击开始接龙即可
3.  第一次接龙之后，出现的成语列表，可以直接点击后面的 接下一个 按钮继续往下接龙
